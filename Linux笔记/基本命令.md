# Linux 基本命令
##### 查看进程：
```
ps -ef|grep python
```
##### 杀死进程
```
kill -9
```
##### 打印log
```
tail -f logs/diff_process.log
```
##### 查找文件
```
find / -name ok.txt
```
##### 查看文件行数
```
wc
- c 统计字节数。
- l 统计行数。
- w 统计字数。
```
##### 查看查找字段上下多少行
```
grep -C 5 foo file 显示file文件里匹配foo字串那行以及上下5行
grep -B 5 foo file 显示foo及前5行
grep -A 5 foo file 显示foo及后5行
```
